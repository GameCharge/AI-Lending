ИИ-Анкета24 — HR-платформа 
Современный лендинг для сервиса умных опросов сотрудников. В основе — высокая конверсия, плавный UX и молниеносная работа на FastAPI.

Особенности

AI-Driven Analytics: Концепция автоматического анализа вовлеченности (eNPS).
High Performance: Бекенд на FastAPI с асинхронной обработкой заявок.

Premium UI/UX:
Интерактивная карусель отзывов с поддержкой свайпов на мобильных устройствах.
Адаптивная верстка.
Сложные CSS-анимации (параллакс мокапов, Intersection Observer для появления контента).

API Методы

POST /api/lead — Прием заявок с формы. 


Ключевые преимущества и уникальные решения проекта

slowapi (rate limiting) — защищает от спама и DDoS, ограничивая количество запросов по IP.

ASGI + FastAPI — асинхронная архитектура позволяет обрабатывать множество запросов одновременно без блокировок, что даёт высокую производительность под нагрузкой.

Honeypot-защита — простой и незаметный для пользователя способ отсеивать ботов без капч и ухудшения UX.

Pydantic-валидация — строгая проверка входных данных ещё до выполнения бизнес-логики, снижает количество ошибок и повышает безопасность.

Vanilla JS + Intersection Observer — минимальный frontend без тяжёлых фреймворков, быстрая загрузка и анимации без лишней нагрузки на браузер.
